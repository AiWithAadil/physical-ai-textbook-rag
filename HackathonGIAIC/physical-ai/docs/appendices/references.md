# References

## Foundational Textbooks

### Robotics and Control

1. **"Introduction to Robotics: Mechanics and Control"** by John J. Craig
   - Comprehensive coverage of kinematics, dynamics, and control
   - Industry standard for robotic manipulation

2. **"Probabilistic Robotics"** by Sebastian Thrun, Wolfram Burgard, Dieter Fox
   - SLAM, localization, and uncertainty handling
   - Foundational for autonomous navigation

3. **"Planning Algorithms"** by Steven M. LaValle
   - Path planning and motion planning
   - Free online: http://planning.cs.uchicago.edu/

### Computer Vision

4. **"Computer Vision: Algorithms and Applications"** by Richard Szeliski
   - Comprehensive vision fundamentals
   - Free online version available

5. **"Deep Learning for Computer Vision"** by Rajagopal Anandakumar
   - Modern deep learning approaches
   - Practical applications

### Machine Learning and AI

6. **"Deep Learning"** by Ian Goodfellow, Yoshua Bengio, Aaron Courville
   - Comprehensive deep learning theory and practice
   - Free online: https://www.deeplearningbook.org/

7. **"Reinforcement Learning: An Introduction"** by Richard Sutton and Andrew Barto
   - Foundational RL concepts
   - Practical algorithms

## Research Papers

### SLAM and Localization

- **ORB-SLAM**: Rublee et al., "ORB-SLAM: a versatile and accurate monocular SLAM system" (IEEE Trans. Robotics, 2015)
- **Visual SLAM**: Thrun et al., "The Graph SLAM Tutorial" (IEEE Robotics & Automation Magazine, 2010)

### Object Detection

- **YOLO**: Redmon et al., "You Only Look Once: Unified, Real-Time Object Detection" (CVPR, 2016)
- **Faster R-CNN**: Ren et al., "Faster R-CNN: Towards Real-Time Object Detection with Region Proposal Networks" (NIPS, 2015)
- **Mask R-CNN**: He et al., "Mask R-CNN" (ICCV, 2017)

### Reinforcement Learning

- **DQN**: Mnih et al., "Human-level control through deep reinforcement learning" (Nature, 2015)
- **Policy Gradient**: Sutton et al., "Policy Gradient Methods for Reinforcement Learning with Function Approximation" (NIPS, 1999)

### Manipulation

- **Grasping**: Sahbani et al., "Robotic Grasp Planning: State of the Art" (Robot. Auton. Syst., 2012)
- **Dextrous Manipulation**: Andrychowicz et al., "Learning Dexterous In-Hand Manipulation" (RSS, 2019)
